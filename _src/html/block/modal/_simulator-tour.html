Simulator Tour

  • Features:

    1. View your test settings like test time, passing score. Note: test time & passing score can be changed. Currently these are set to actual test conditions.
    2. View all exams available.
    3. View your exam history - view all test results from your previous practice tests, once you have taken any. If you have any unfinished exams, you can resume them from here, or you can also delete unwanted exams.
    4. Practice exam mode - when this mode is used, you can see correct answers and explanations for each question.
    5. Virtual exam mode - in this mode you will get results only at the end of your exam. This is a simulation of an actual test environment.

  • Features:

    1. Timer/count down - indicates how much time you have left. It starts automatically once you launch your exam.
    2. Number of questions - indicates the number of questions in this test. Each question has several options. Just tick a box with the correct answer.
    3. Previous/next button navigation - if you aren't sure about some questions you can return back using 'previous' and 'next' buttons.
    4. End exam - you can end your exam anytime and then resume it from your main dashboard area.

  • Features:

    1. Correct answers with explanations - if you are not sure about a correct answer, just click on the answers and explanation link which will reveal a correct answer including an explanation.

  • Features:

    1. Mark for review - if for whatever reason you would like to review any question, simply mark it for review then review it by clicking on 'review marked' button.
    2. Make/read notes. This feature allows you to create your own notes and then read them later.

  • Features:

    1. Review market - this feature will display all questions available in your test plus any questions you marked for future review.

  • Features:

    1. The time elapsed.
    2. How many questions your answered correctly or incorrectly.
    3. Your passing score.

  • Features:

    1. Test time - this can be changed according to your needs, however 90 minutes is the standard time.
    2. Passing score - in order to pass your test, your passing score must be at least 85% or above.
    3. Questions in test - if you prefer to practice less questions at once then this option allows you to do so.
    4. Always show - if you always show, answers and explanations to all questions you will be always visible.

  • Features:

    1. Practise questions in random order.
    2. You can also select answers to questions in random order.
    3. Here you can set if answers and exlanations should be displayed automatically, or only when selected.

  • Features:

    1. Select this option in order to see all questions from all exam topics.
    2. This option allows you to see questions by type.
    3. If you want to see questions only from certain topics use this option.

  • Features:

    1. You can change appearance of questions and answers like font, font size, style and colours to the way you want it to be displayed.
    2. You can set different prompt messages: on closing of program, when removing items from exam history list and on finishing test in progress.

  • Features:

    1. If you are behind secure networks, you can set host and port in case of using proxy servers. You can also set username and password, if your network requires authentication.
